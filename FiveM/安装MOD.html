QuantV画质增强

官方源 捐赠10美元

MEGA分流

第一步

\QuantV FiveM 2024-1-23.zip\Part1\FiveM.app\mods\里的文件放在\FiveM Application Data\mods\里

第二步

\QuantV FiveM 2024-1-23.zip\Part2\ReShade\Part_2-1\FiveM.app\plugins\里的文件放在\FiveM Application Data\plugins\里

第三步

\QuantV FiveM 2024-1-23.zip\Part2\ReShade\Part_2-2\NUI in-process GPU off (default)\FiveM.app\plugins\的文件放在\FiveM Application Data\plugins\里

第四步

  • 游戏里按Home键开启菜单，可以在游戏里自行改键
  • 建议只安装ReShade，电脑配置高可以安装ENB或者ReShade+ENB
  • 更详细的官方安装教程请查看压缩包内的README.html文件

Remastered画质增强

官方源（FiveM-client版）

MEGA分流

第一步

vremastered.rpf文件放在\FiveM Application Data\mods\里

第二步

default.fxc文件放在\FiveM Application Data\citizen\shaderz\win32_40_final\里

圣安地列斯树林

官方源（FiveM-client版）

MEGA分流

第一步

fosa_n.rpf、fosa_s.rpf文件放在\FiveM Application Data\mods\里

第二步

default.fxc文件放在\FiveM Application Data\citizen\shaderz\win32_40_final\里

（和Remastered画质增强的default.fxc文件一模一样，只用安装一次）

Roads of Europe

官方源（FiveM-client版）

MEGA分流

RoadsOfEuropeDE.rpf文件放在\FiveM Application Data\mods\里

⚠️为了防止远处的模型有黄色虚线纹理，请在 nvidia 控制面板中将各向异性过滤强制为 16x

LambdaMenu修改器

下载lambdamenu.zip后解压，接着

第一步

LambdaMenu.asi文件放在\FiveM Application Data\plugins\里

第二步

lm-config.xml文件放在\FiveM Application Data\里

游戏里F5打开菜单，F6打开穿越模式，F7打开导航点传送，可以打开xml自行改键

人物模型替换

F5 - Player - Player Skin - 第二页Spawn Player Model By Name

或者你也可以使用Menyoo来替换，只需把以下的文字填入Menyoo的PedList里，教程此处不赘述，自行理解或者搜索

⚠️但是，如果你使用Menyoo，需要将menyooConfig.ini里的部分设置调整如下：

manual respawn = true

DeathModelReset = false

否则在玩图时会触发无限复活
