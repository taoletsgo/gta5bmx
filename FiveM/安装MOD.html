字体mod🔠

在差事选择页面显示中文/日文字符，不再有讨厌的方框！

下载地址(加入discord获取)

第一步

\FiveM Application Data\文件夹下新建\addons文件夹

第二步

将font.rpf复制进去

  • 目前受支持的游戏设置语言

“en”、“ja”、“zh-cn”、“fr”、“de”、“it”、“ru”、“pt”、“pl”、“pt”、“es”、“es-mx” 

  • 目前不支持的游戏设置语言

“zh”、“kr”

（如果服务器里有很多韩国人/台湾人/新加坡人我会再次更新，但目前看来不需要支持）

画质mod🏙️

QuantV画质增强

官方源 捐赠10美元

MEGA分流

第一步

\QuantV FiveM 2024-2-21.zip\Part1\FiveM.app\mods\里的文件放在\FiveM Application Data\mods\里

第二步

\QuantV FiveM 2024-2-21.zip\Part2\ReShade\Part_2-1\FiveM.app\plugins\里的文件放在\FiveM Application Data\plugins\里

第三步

\QuantV FiveM 2024-2-21.zip\Part2\ReShade\Part_2-2\NUI in-process GPU off (default)\FiveM.app\plugins\的文件放在\FiveM Application Data\plugins\里

第四步

  • 游戏里按Home键开启菜单，可以在游戏里自行改键
  • 建议只安装ReShade，电脑配置高可以安装ENB或者ReShade+ENB
  • 更详细的官方安装教程请查看压缩包内的README.html文件

修改器🛠️

Menyoo修改器

🗣️对于习惯英文界面的玩家，可去MenyooGithub下载

🗣️对于习惯中文界面的玩家，可下载此汉化版本MenyooMenu(Chinese).7z

第一步

Menyoo.asi文件放在\FiveM Application Data\plugins\里

第二步

menyooStuff整个文件夹放在\FiveM Application Data\data\cache\subprocess\里

游戏里F9打开Spooner模式（类似于制作器），F10打开菜单（首次打开需要按两次），F11打开noclip自由摄像机模式（默认关闭），可以打开\menyooStuff\menyooConfig.ini自行改键

⚠️如果你是从Github下载的Menyoo，对应的默认按键分别为F9（Spooner）、F8（Menu）、F3（noclip），中文文本的翻译文件可从汉化版本\menyooStuff\Language\里提取

⚠️如果你是从Github下载的Menyoo，请务必将\menyooStuff\menyooConfig.ini里的部分设置调整如下，DeathModelReset = false，不然你会触发无限复活（类型A）

LambdaMenu修改器

下载lambdamenu.zip后解压，接着

第一步

LambdaMenu.asi文件放在\FiveM Application Data\plugins\里

第二步

lm-config.xml文件放在\FiveM Application Data\里

游戏里F5打开菜单，F6打开穿越模式，F7打开导航点传送，可以打开xml自行改键
