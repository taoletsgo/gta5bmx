vMenu

一个外挂菜单，键盘M键打开

这里面可以生成角色/武器/载具/传送等，详细功能自己拍照翻译来熟悉

Player Related Options

改无敌/获得武器/生成角色/捏脸等

Vehicle Related Options

载具设定/生成载具/保存载具等

World Related Options

改天气/改时间

Misc Settings

主要介绍一个里面的传送功能（Misc Settings - Teleport Options）

  • Teleport to Waypoint 传送到导航点，请先在地图上标记
  • Teleport to Coords 传送到坐标，依次输入x，y，z的坐标
  • Teleport Locations 已保存的坐标
  • Save Teleport Location 保存当前坐标位置并命名，然后在Teleport Locations里能找到

DarkRP_Racing

利用vMenu传送到坐标x=0，y=0，z=70

按E打开，点击Create Game，会弹出下面的界面

左边是开图，右边是作图

开图

模式选择Parkour，这个模式可以半途加入，其他设置和GTA online差不多

作图

命名尽量标准，不然查询麻烦

  • Title 输入标题（简体中文或者英文）
  • Description 输入简介
  • Length 检查点总距离要大于1公里才能保存
  • Route Type 选择Point to Point（点对点），没有特殊需求，不要选择Laps（绕圈）
  • Map Type 设置为作者名字或者缩写，比如ioio/RA/nape777/pino等
  • Image 封面图可以直接回车留空，也可以从Rockstar job页面右键复制图片网址（举例）并粘贴
  • Force Vehicle 设定为BMX
