QuantV画质增强

官方源 捐赠10美元

MEGA分流

第一步

\QuantV FiveM 2023-12-30.zip\Part1\FiveM.app\mods\里的文件放在\FiveM Application Data\mods\里

第二步

\QuantV FiveM 2023-12-30.zip\Part2\ReShade\Part_2-1\FiveM.app\plugins\里的文件放在\FiveM Application Data\plugins\里

第三步

\QuantV FiveM 2023-12-30.zip\Part2\ReShade\Part_2-2\NUI in-process GPU off (default)\FiveM.app\plugins\的文件放在\FiveM Application Data\plugins\里

第四步

  • 游戏里按Home键开启菜单，可以在游戏里自行改键
  • 建议只安装ReShade，电脑配置高可以安装ENB或者ReShade+ENB
  • 更详细的官方安装教程请查看压缩包内的README.html文件

Remastered画质增强

官方源（FiveM-client版）

MEGA分流

第一步

vremastered.rpf文件放在\FiveM Application Data\mods\里

第二步

default.fxc文件放在\FiveM Application Data\citizen\shaderz\win32_40_final\里

圣安地列斯树林

官方源（FiveM-client版）

MEGA分流

第一步

fosa_n.rpf、fosa_s.rpf文件放在\FiveM Application Data\mods\里

第二步

default.fxc文件放在\FiveM Application Data\citizen\shaderz\win32_40_final\里

（和Remastered画质增强的default.fxc文件一模一样，只用安装一次）

Roads of Europe

官方源（FiveM-client版）

MEGA分流

RoadsOfEuropeDE.rpf文件放在\FiveM Application Data\mods\里

⚠️为了防止远处的模型有黄色虚线纹理，请在 nvidia 控制面板中将各向异性过滤强制为 16x

LambdaMenu修改器

下载lambdamenu.zip后解压，接着

第一步

LambdaMenu.asi文件放在\FiveM Application Data\plugins\里

第二步

lm-config.xml文件放在\FiveM Application Data\里

游戏里F5打开菜单，F6打开穿越模式，F7打开传送，可以打开xml自行改键

如果出现莫名其妙的bug，请删除"C:\Users\用户名\AppData\Roaming\LambdaMenu"里的文件再进游戏
