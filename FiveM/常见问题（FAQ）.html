2.创建差事菜单里怎么只有一个差事？

回车键进入第二层级菜单后，再根据作者（标签）选择地图，或者输入关键词搜索地图

3.多个玩家在同一场差事里，BMX怎么明显被加速/被减速/有碰撞？

请关闭碰撞/减速保护/尾流/Catch-Up这四个选项

4.不想要布娃娃系统

使用内置修改器开启no ragdoll模式

5.BMX从高空落地时，人物怎么会容易从单车上摔落？

这就是正常的物理特性

6.新加入一个差事，怎么有时候没办法接触检查点，或者摔落后丧失布娃娃系统且无法重生？

游玩上一个差事时，在死亡提示页面时或者重生后秒退了，如果需要退出，请在重生后等待3s左右，否则下一个差事你需要反复进入两次，才可以正常游玩

7.在部分差事里，BMX怎么重生位置在地面？

由于检查点在红/绿集装箱上，导致重生位置判定错误，管理员在后台手动调整该差事的配置后，即可恢复正常

8.OBS和锁帧软件RTSS一块用，OBS怎么没有画面？

RTSS - Setup - General - User Microsoft Detours API hooking（勾选），并修改数值为5000
