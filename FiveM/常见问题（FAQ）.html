2.创建竞速菜单里怎么只有一个竞速差事？

回车键进入第二层级菜单后，再根据作者（标签）选择地图，或者输入关键词搜索地图

3.如何关闭倒计时（did not finish）？

在创建比赛时，修改倒计时（DNF）为“关闭（no）”

4.多个玩家在同一场竞速差事里，BMX怎么明显被加速/被减速/有碰撞？

在创建比赛时，关闭碰撞/减速保护/尾流/Catch-Up这四个选项

5.新加入一个竞速差事，怎么有时候没办法接触检查点，以及无法重生？

因为你在游玩上一个竞速差事时，处于死亡提示页面时或者重生后，你秒退了。如果需要退出，请在重生后等待3s左右，否则下一个竞速差事你需要反复进入两次，才可以正常游玩

6.在载具（比如BMX）附近生成新的BMX，怎么控制参数与碰撞参数变得很奇怪？

切记在生成新的BMX时，不要与任何碰撞体（比如BMX）接触；也不要在跑步时生成BMX

7.在竞速差事里，BMX从高空落地时，人物怎么会容易从单车上摔落？

这就是正常的物理特性，且不会更新。

当前摔落级别设定：roll的绝对值小于45度时，不会触发布娃娃，也就是说低速撞击或中等程度磕绊不会摔落。你可以输入/rot并查看F8控制台来debug

欢迎提摔落级别相关的改进，但，高空落地和高速撞击的设定不会修改

8.在竞速差事里，为什么游戏会陷入无限复活？

  • 无限复活（类型A）
触发原因：使用了内置修改器menyoo，在竞速里死亡后，会变成故事模式的角色（比如麦克），然后陷入无限复活
解决方案：需要将menyooConfig.ini里的设置调整如下，DeathModelReset = false
  • 无限复活（类型B）
触发原因：在竞速里更换mod人物模型，导致验证机制判定你不在载具里，然后陷入无限复活
解决方案：请在加载竞速差事前替换人物模型

9.在部分竞速差事里，BMX怎么重生位置在地面？

由于检查点在红/绿集装箱上，导致重生位置判定错误，管理员在后台手动调整该竞速差事的配置后，即可恢复正常

10.在部分竞速差事里，部分道具怎么缺失了？

因为有一部分道具FiveM不支持，或者道具是新版本DLC的。如果你发现了道具缺失，可以帮我在https://forge.plebmasters.de/objectss上找到确切的名字，然后我将其模型流式传输到客户端

11.如何锁定FPS的数值，以及自定义绑定按键切换帧数？

正如安装本体中所描述的方法，使用RTSS软件进行锁帧，进阶使用请自行探索

12.OBS和锁帧软件RTSS一块用，OBS怎么没有画面？

RTSS - Setup - General - User Microsoft Detours API hooking（勾选），并修改数值为5000
