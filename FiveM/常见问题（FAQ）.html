常见问题⁉️

Q：创建差事菜单里怎么只有一个差事？

A：回车键进入第二层级菜单后，再根据作者（标签）选择地图，或者输入关键词搜索地图

 

Q：多个玩家在同一场比赛里，BMX怎么明显被加速/被减速/有碰撞？

A：请关闭碰撞/减速保护/尾流/Catch-Up这四个选项

 

Q：BMX从高空落地时，怎么会容易摔死？

A：这就是正常战局里的物理特性

 

Q：在部分差事里，BMX怎么重生位置在地面？

A：由于检查点在红/绿集装箱上，导致重生位置判定错误，管理员在后台手动调整配置后，即可恢复正常

 

Q：OBS和锁帧软件RTSS一块用，OBS怎么没有画面？

A：RTSS-Setup-General-User Microsoft Detours API hooking（勾选），并修改数值为5000
