# **砸后轮**🕹️

## **砸后轮**

先稍微后拉单车，然后空中按跳蓄力，最后双轮或前轮落地前推+刹车+跳

![](/教程/砸后轮/1.砸后轮.gif)

-   低帧双轮落地砸，高帧前轮落地砸

### **按键**🎮

```plaintext
键盘：方向按键+空格+Q
手柄：左摇杆+RB/R1+LT/L2
```

## **反动砸后轮**

带上左右方向的砸后轮，起名参考了反动跳

![](/教程/砸后轮/2.反动砸后轮.gif)
