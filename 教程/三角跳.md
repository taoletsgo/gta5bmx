# **三角跳**🕹️

前轮在斜面，后轮在地面，形成三角关系

![](/教程/三角跳/2.坡度三角跳.gif)

-   低帧更容易成功，遇到三角跳请立刻锁30~45帧
-   起跳时先微微后拉，是为了砸的更有力
-   三角跳的高度和距离与速度也有关系

### **按键**🎮

```plaintext
键盘：方向按键+空格
手柄：左摇杆+RB/R1
```
